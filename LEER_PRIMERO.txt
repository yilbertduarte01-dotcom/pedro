PASO 1: INICIALIZAR LA BASE DE DATOS
================================================================================

Después de descargar el proyecto, DEBES limpiar y recrear la base de datos.

1. Abre tu navegador y accede a:
   http://localhost/liceo/limpiar-bd.php

2. Verás una página de advertencia. Lee cuidadosamente.

3. Marca el checkbox: "Entiendo que todos los datos serán eliminados y acepto continuar"

4. Haz clic en el botón: "🗑️ Limpiar Base de Datos"

5. Se eliminará todo y se recrearán automáticamente todas las tablas
   con la estructura correcta.

6. Serás redirigido al inicio en 2 segundos.

7. ¡LISTO! Ya puedes usar el sistema sin problemas.

PREGUNTAS FRECUENTES
================================================================================

P: ¿Qué hago si no puedo crear reuniones?
R: Verifica que hayas iniciado sesión como Profesor o Administrador.
   Si aún tienes problemas, limpia la BD nuevamente.

P: ¿Por qué algunos módulos no carga?
R: Accede a limpiar-bd.php para resetear completamente la base de datos.

P: ¿Puedo cambiar las contraseñas?
R: Las contraseñas están en login.php. Puedes editarlas directamente.

P: ¿Cómo agrego más secciones?
R: En el módulo Profesor, hay un botón "Crear Sección".
   En Reuniones, selecciona de la lista predefinida.

P: ¿Se guardan todos los datos?
R: Sí, todo se guarda en la base de datos MySQL.
