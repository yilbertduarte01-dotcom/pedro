# Plantilla de Inscripción de Representante - VERSIÓN MEJORADA
## Guía Rápida de Uso

### Acceso Rápido
1. Entra al módulo **Representante** en el dashboard
2. En la pestaña principal "Información Institucional"
3. Busca la sección azul: **"Plantilla de Inscripción de Representante"**
4. Haz clic en **"⬇️ Descargar PDF"**

### Lo Que Sucede
El sistema abre una página HTML optimizada para impresión que contiene el formulario completo editable en el navegador.

### Opciones para Descargar/Guardar

#### Opción 1: Guardar como PDF (Recomendado)
1. Una vez abra el formulario, verás un botón azul **"🖨️ Imprimir / Guardar como PDF"**
2. Haz clic en el botón
3. En la ventana de impresión, selecciona **"Guardar como PDF"** en lugar de impresora
4. Elige la carpeta donde guardar
5. El archivo se guardará como `Plantilla_Inscripcion_YYYY-MM-DD.pdf`

#### Opción 2: Imprimir Físicamente
1. Haz clic en **"🖨️ Imprimir / Guardar como PDF"**
2. Selecciona tu impresora
3. Haz clic en **"Imprimir"**
4. Completa el formulario a mano

#### Opción 3: Editar en Navegador e Imprimir
1. Puedes llenar el formulario usando el navegador (aunque los campos están optimizados para impresión)
2. Luego usar "Guardar como PDF"

### Contenido del Formulario

#### Sección 1: Datos del Representante
- Nombre Completo
- Cédula/ID
- Teléfono
- Correo Electrónico
- Dirección
- Ocupación

#### Sección 2: Datos del Representado (Estudiante)
- Nombre Completo
- Cédula
- Fecha de Nacimiento
- Nivel (1ro, 2do, 3ro, etc.)
- Sección (A, B, C, etc.)
- Género
- Talla de Uniforme

#### Sección 3: Antecedentes de Salud
- Alergias
- Medicamentos que toma regularmente

#### Sección 4: Información de Emergencia
- Nombre del Contacto de Emergencia
- Teléfono de Emergencia
- Relación con el estudiante

#### Sección 5: Autorización
- Declaración de consentimiento informado
- Espacio para firma del representante
- Espacio para fecha

### Instrucciones Después de Descargar

#### Si Descargaste como PDF
1. Abre el PDF con Adobe Acrobat Reader o similar
2. Completa los campos que tengan líneas para llenar
3. Firma digitalmente o imprime y firma a mano
4. Guarda una copia para tus registros
5. Envía al correo o lleva a la institución

#### Si Imprimiste Directamente
1. Completa con bolígrafo azul o negro
2. Escribe claramente en los espacios asignados
3. Firma al pie del documento
4. Lleva el original a la institución

### Solución de Problemas

**Problema: El botón no abre nada**
- Verifica que hayas iniciado sesión correctamente
- Prueba con otro navegador
- Limpia el cache del navegador

**Problema: El formulario se ve mal**
- Usa la opción de impresión (Ctrl+P)
- Cambia de navegador (Chrome, Firefox, Safari, Edge)
- Asegúrate de que JavaScript esté habilitado

**Problema: No puedo firmar digitalmente**
- Adobe Acrobat Reader permite firmas digitales
- También puedes imprimir y firmar a mano
- Toma foto de la firma escaneada y adjunta

**Problema: Se cortó la impresión**
- En la ventana de impresión, ajusta márgenes a mínimo
- Reduce el zoom si es necesario
- Asegúrate de que esté en tamaño carta (A4)
